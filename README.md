## Installation

Clone this repository

```
git clone https://github.com/davidtkramer/factory-workshop.git
```

Install dependencies

```
bundle install
```

Checkout the first branch

```
git checkout 1-intro
```

Run the tests in watch mode

```
bundle exec guard
```
